Service Ticket Classification

Created a machine learning model that can classify tickets using KNN and assign to the respective teams. Additionally, a webpage was created for the ease of data scientists to upload data and train the model. Supported drag and drop tool to add features in training process. This let users with no knowledge of machine learning to train and see the insights/output of the data uploaded from any device.

Technologies Used: Python, KNN Classification Algo, HTML, CSS, Flask, Docker, Nodejs

Objective: To create a web application that can be used to train models and predict the classes for future data.

A web-application enabled with capabilities to drag and drop with some of the Supervised-Learning Machine Learning abilities. Through which the user is allowed to upload data and specify the attributes to consider for training and allowed to train. Once the training is done, the model is used to predict the new classes of the test data/unseen data is being uploaded. This application was developed to support more of the machine learning abilities with the help of UI and let users with not enough knowledge on Machine learning use it.

The whole application was written in Python and is hosted as a server using Flask on a Docker container. Every request is passed on to the Flask Server via Nodejs framework which handles the API Calls.
