Contact Display Application

CRUD operations on MySQL database from a user interactable webpage developed using Django. Used ORM to retrieve and insert data into the database.

Technologies Used: Python, Django Web Framework, MYSQL Database, HTML, CSS, MDN Bootstrap, Javascript, jquery

Objective: To perform Create, Read, Update and Delete (CRUD) operations on MySQL database via an UI based application.

This was one of my projects in the Database Design course at UTD. The web application is created using Python and Django Framework. Web-app lets you add new contact contact details of a person. Also, it lets you delete or modify the record in the databse directly via the UI. One significant feature enabled in the web-app is that the user can search the contatc details in single line search similar to the ones like Google, Bing search. Using Django Object Realtional Mapper(ORM), the application was allowed to pull the database information from MySQL to Python as classes.

I personally learned a lot from this project. For the first time, I laid my hands on Django, with the help of Python knowledge I was able to grasp the details and nuances of the Django framework and it let me complete the project in a very less days of timeframe.
