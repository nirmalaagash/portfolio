Object Identification and Validation

Implemented an object detection model for a consumer goods customer that validates the purchase of the product by a user and delivers rewards. Provides ease of access for the users to get rewards with no latency. Increased the rate of validation process by 10% and the number of users verifying increased by 12% from the customer reports.

Technologies Used: Python, Tensorflow Object Detection API, TF lite, HTML, CSS, Pre-trained model(ssd-mobilenet)

Objective: To identify and verify the purchase of the product from the webcam/front camera.

This was for one of the biggest consumer goods client. The requirement was to identify if the customer has purchased the product and upon the verification of the purchase, follow-up duties are performed for the customer.

The project was developed in Python using Tensorflow Object Detection API which requires a huge set of images of the product that is to be identified. It was trained upon the pre-traiend model ssd-mobilenet-v2.0. After training, the model was used in prediction of the object from the webcam/front camera of the mobile and once verified it is passed onto the next chore that is to be performed. The main obstacle that was faced during the training phase of the model was identifying the object apart from the competitors' products. This really spiked up my knowledge in the field of deep learning a bit and made me explore much.
