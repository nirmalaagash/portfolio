What I worked on

  • Incident Management Chatbot

    Established an intelligent chatbot that replaces human involvement by 80% that connects with other API services to resolve the ticket. Unknown manual resolving actions are recorded and suggested for similar issues in future occurring tickets. The time to resolve similar ticket that had occurred earlier has been decreased by nearly 60% without human intervention.

  • Object Identification and Validation

    Implemented an object detection model for a consumer goods customer that validates the purchase of the product by a user and delivers rewards. Provides ease of access for the users to get rewards with no latency. Increased the rate of validation process by 10% and the number of users verifying increased by 12% from the customer reports.

  • Service Ticket Classification

    Created a machine learning model that can classify tickets using KNN and assign to the respective teams. Additionally, a webpage was created for the ease of data scientists to upload data and train the model. Supported drag and drop tool to add features in training process. This let users with no knowledge of machine learning to train and see the insights/output of the data uploaded from any device.

  • Golden Record Identification

    Similar records with missing attributes from different sources are combined using HiveQL and the combined golden record is moved into PostgreSQL Database. This lets customers to recover 35% of lost records information due to errors from sources.

  • Contact List Display Application

    CRUD operations on MySQL database from a user interactable webpage developed using Django. Used ORM to retrieve and insert data into the database.

  • Relational Database File Management

    Built a database file management system from scratch that stores in the data in binary file with multi indexing of B-tree implementation. Supports a Shell Window to interact with the database. Aids basic operations like Select, Show, Insert, Create like the MySQL or SQLite commands.

  • Data Analysis and Linear Regression using R

    Pre-processing and diagnostics of the dataset ("California Housing Prices") is performed using Linear Regression in R. It gives the practical understanding of the concepts involved in the pre-training of the model and patterns in the data features are analysed by plotting graphs.
